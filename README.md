##  How to run

### API ( cd /backend-api )

> **Install Dependencies**
>
> $ yarn install
> 
> **Run Aplication**
>
> $ yarn start

### Web ( cd /frontend-web )

> **Install Dependencies**
>
> $ yarn install
> 
> **Run Aplication**
>
> $ yarn start

### Mobile  ( cd /mobile )

> **Install Dependencies**
>
> $ yarn install
> 
> **Run Aplication**
>
> $ yarn start
